Утилита поставляется как кроссплатформенный бинарный исполняемый файл cyk.
https://github.com/grikonst/analytics_manager/blob/cc682f5f5e80a9f25f010ee58ea6bc48c8ea3c07/cyk && chmod +x cyk

в Терминале, выполнить команду:
sh cyk
введите классический пароль для начала работы с утилитой.  

Основное меню утилиты:
1) Управление агентами аналитики (scanner,bags)                    
2) Управление видеопотоками (добавление,запуск\остановка,удаление)                         
3) Запуск записи потоков StreamRecorder (видеозапись mp4)
4) Конфигурация системы (редактировать шаблон аналитики)                             
5) Диагностика и мониторинг (ffmpeg,sysreports)                        
6) Получить версии агентов аналитики (архив с образами версий агентов)

Структура и файлы конфигов хранятся в $HOME/.stream_manager/*

Структура и файлы конфигов в $HOME/.stream_manager/*
- config               | Основные настройки
- template_req.json    | Шаблон аналитик JSON
- agents/              | Настройки агентов scanner и bags
- agents/releases      | Образы агентов scanner и bags 
- cams_list/           | Файлы списков видеопотоков 
- logs_archive/        | Архивы логов агентов
- recorder/			   | ВидеоЗаписи потоков с камер(.mp4)
- reports/			   | Отчёты диагностики и состояния ОС
