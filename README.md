Утилита поставляется как кроссплатформенный бинарный исполняемый файл cyk.
в Терминале, выполнить команду:
./cyk
введите классический пароль для начала работы с утилитой.

Основное меню утилиты:
1) Управление агентами аналитики (Scanner, Bags)                    
2) Управление видеопотоками                         
3) Конфигурация системы                             
4) Диагностика и мониторинг (Ffmpeg)                        
5) Получить релизы агентов аналитики (*.zst)
6) Включить StreamRecoder для записи видеопотоков (recoder.tar)

Структура и файлы конфигов в $HOME/.stream_manager/*
- config               | Основные настройки
- template_req.json    | Шаблон аналитик JSON
- agents/              | Настройки агентов
- cams_list/           | Файлы списков видеопотоков
- logs_archive/        | Архивы логов агентов
